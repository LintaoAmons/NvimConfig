# Keybindings Configuration

- `no-leader-keymappings` 放了不是以 `<leader>` 打头的快捷键配置
- `which-key` 放了所有以 `<leader>` 打头的快捷键配置，并且加上了说明
- `telescope` 有一些 telescope 相关的快捷键配置
